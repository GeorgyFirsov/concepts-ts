\infannex{diff}{Compatibility}

\rSec1[diff.iso]{\Cpp and ISO \Cpp 2017}

\pnum
This subclause lists the differences between \Cpp
and ISO \Cpp 2017, by the chapters of this document.

\rSec2[diff.lex]{Clause~\ref{lex}: lexical conventions}

\ref{lex.key}
\change New keywords.\\
\rationale Required for new features.
The \tcode{requires} keyword is added
to introduce constraints through a \grammarterm{requires-clause} or
a \grammarterm{requires-expression}. The \tcode{concept} keyword is
added to enable the definition of concepts (\ref{temp.concept}).
\effect
Valid ISO \Cpp 2017 code using \tcode{concept} or \tcode{requires}
as an identifier is not valid in this International Standard.
